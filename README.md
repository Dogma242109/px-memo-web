# px memo web

低解像度キャンバスで描いて、ドット風に拡大表示するメモアニメーションWebアプリです。

## 起動方法

Python 3でローカルサーバーを起動します。

```powershell
python app.py
```

`python` が認識されない場合は、VS CodeでPythonインタープリターを選ぶか、Python本体のパスを指定して起動します。

```powershell
C:\msys64\mingw64\bin\python.exe app.py
```

ブラウザで以下を開きます。

```text
http://127.0.0.1:8000/
```

終了するときは、ターミナルで `Ctrl+C` を押します。
