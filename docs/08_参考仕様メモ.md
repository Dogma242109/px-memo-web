# 参考仕様メモ

## 目的

うごくメモ帳3D系の仕様を参考にしつつ、このプロジェクトで採用するもの、後回しにするものを整理する。

完全再現ではなく、低解像度、制限された色、フレームアニメーション、軽い操作感を参考にする。

## 画面とキャンバス

- 解像度は320×240px
- 実際の描画領域は310×230px
- 出力方法によってサイズや周辺の縁が変わる場合がある

### このプロジェクトでの判断

- 採用: 内部描画領域を310×230pxにする
- 採用: 表示倍率3倍で、PC画面上では930×690pxとして扱う
- 見送り: 出力時の周辺縁やサイズ変動の再現

## 色と背景

- 色は黒、白、赤、青、緑、黄の6色
- 背景は6色または方眼を選べる
- 背景色を変えてもペン色は変わらない
- 1つのレイヤーでは2色まで、全体で最大6色まで使える

### このプロジェクトでの判断

- 一部採用: 初期は自由な色選択を残す
- 将来候補: 6色パレットモードを追加する
- 将来候補: 背景色と方眼背景を選べるようにする
- 将来候補: レイヤーごとに使える色を制限する

## ペンとブラシ

- ペン先は複数種類ある
- パターンブラシやメモリーブラシがある
- 消しゴムにも複数種類がある
- ペン先の太さを変更できる
- 図形作画として円、四角形、直線を描ける

### このプロジェクトでの判断

- 採用: 初期ペンはアンチエイリアスなしの四角ブラシ
- 採用: サイズごとのドットマスクブラシを追加する
- 採用: +形や疑似丸形のペン先を追加する
- 採用: ベタに加えて、点々、市松、斜格子、縦線、横線、網、黒市松、穴あきの8種類のパターンブラシを追加する
- 採用: パターンブラシはペン位置ではなくキャンバス絶対座標を基準にする
- 採用: ペン先形状とパターンは別々に選べるようにする
- 将来候補: 直線、四角形、円ツールを追加する
- 見送り: メモリーブラシなどの高度な特殊ブラシ

## レイヤー

- レイヤーは3枚
- レイヤーの入れ替え、統合、非表示ができる
- 編集レイヤーを切り替えられる
- レイヤーごとに3D深度を指定できる

### このプロジェクトでの判断

- 将来候補: 3レイヤ構成を追加する
- 将来候補: レイヤーの表示/非表示を追加する
- 将来候補: レイヤー統合を追加する
- 見送り: 3D深度

## ページと再生

- ページ切り替えでアニメーションを作る
- 最大999ページまで扱える
- 速度は11段階

| Speed | FPS | 秒/page | 備考 |
| --- | ---: | ---: | --- |
| 0 | 0.2 | 5.000 |  |
| 1 | 0.5 | 2.000 |  |
| 2 | 1 | 1.000 |  |
| 3 | 2 | 0.500 |  |
| 4 | 4 | 0.250 |  |
| 5 | 6 | 0.166 | 4コマ相当 |
| 6 | 8 | 0.125 | 3コマ相当 |
| 7 | 12 | 0.083 | 2コマ相当 |
| 8 | 20 | 0.050 |  |
| 9 | 24 | 0.042 | 1コマ相当 |
| 10 | 30 | 0.033 |  |

### このプロジェクトでの判断

- 採用: 初期はFPSを直接指定できるようにする
- 将来候補: Speed 0-10のプリセット選択を追加する
- 将来候補: ページ数が増えたときのメモリ表示を追加する
- 見送り: 999ページ上限の厳密な再現

## 編集補助

- オニオンスキンは1-4枚まで指定できる
- 前後のページを透かせる
- ピクセル単位の画像シフトができる
- ズーム状態で描ける
- Undo/Redoは最大128回
- 範囲ペンでコピー、切り取り、反転、縁取りができる
- 変形で反転、回転、拡大縮小ができる

### このプロジェクトでの判断

- 将来候補: オニオンスキンを追加する
- 将来候補: Undo/Redoを追加する
- 将来候補: ピクセル単位の画像シフトを追加する
- 将来候補: ズーム編集を追加する
- 見送り: 範囲ペンと滑らかな変形

## テキスト、音声、カメラ、出力

- 横書きテキストを好きな位置に挿入できる
- カメラ画像を取り込める
- 音声と効果音を取り込める
- GIF、ANI、JPEG、MPOなどで出力できる

### このプロジェクトでの判断

- 将来候補: GIF出力
- 将来候補: PNG連番出力
- 見送り: 音声、カメラ、立体視、ANI、MPO

## 初期実装への反映

- 内部描画サイズは310×230px
- 表示倍率は3倍
- 四角ピクセルブラシを使う
- まずは1レイヤ、複数フレーム、FPS直接指定で進める
