# 画面設計

## メイン画面

初期バージョンは、1画面で描画、フレーム管理、再生を行う。

```text
+--------------------------------------------------+
| ヘッダー / 作品名                                |
+--------------------------------------------------+
| ツールバー                                      |
| ペン / 消しゴム / 色 / 太さ / 戻す / やり直し    |
+-------------------------------+------------------+
|                               | フレーム一覧     |
| キャンバス                    |                  |
|                               | [1][2][3][+]     |
|                               |                  |
+-------------------------------+------------------+
| 再生操作                                         |
| 再生 / 停止 / FPS                                |
+--------------------------------------------------+
```

## 主な操作

- ペン
- 消しゴム
- 色選択
- ペンサイズ
- フレーム追加
- フレーム削除
- フレーム複製
- 再生
- 停止
- FPS変更
- 保存
- 読み込み

## UIで大切にすること

- 描画エリアを大きく取る
- フレーム一覧は常に見えるようにする
- 再生操作はわかりやすい位置に置く
- 初期版では機能を増やしすぎない
- PCブラウザで使いやすいことを優先する
