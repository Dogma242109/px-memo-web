# 学習メモ

## API

APIはApplication Programming Interfaceの略。

アプリやプログラム同士がやり取りするための窓口や約束を指す。

OpenAI APIやGemini APIは、外部サービスが提供するAPI。FastAPIで作るAPIは、自分のアプリ用に自分で用意するAPI。

## FastAPI

FastAPIは、PythonでWeb APIを作るためのフレームワーク。

将来的に、ログイン、作品保存、クラウド同期、GIF/動画変換、WebSocket通信などを担当させる候補。

## 単一責任原則

単一責任原則。ひとつの部品には、ひとつの主な責任だけを持たせるという考え方。

## 関心の分離

関心の分離。システム全体を役割ごとに分ける考え方。

このプロジェクトでは、描画、フレーム管理、再生、保存、認証、変換、同期などを分けて考える。
