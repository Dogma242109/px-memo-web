# アーキテクチャ

## 初期構成

初期段階では、フロントエンドのみで動作する構成にする。

```text
ブラウザ
  HTML
  CSS
  JavaScript
  Canvas API
  localStorage / IndexedDB
```

## 初期段階の責任範囲

### フロントエンド

- 画面表示
- ツールバー操作
- Canvas描画
- フレーム管理
- アニメーション再生
- ローカル保存
- ファイル書き出し

### ブラウザ内保存

- アプリ設定の保存
- 作品データの保存

## 将来構成

将来的にバックエンドを追加する場合は、FastAPIを候補とする。

```text
ブラウザ
  |
  | HTTP / WebSocket
  v
FastAPI
  |
  v
データベース / ファイル保存領域
```

## 将来のバックエンド責任範囲

- ユーザー登録
- ログイン
- 認証状態の管理
- 作品データの保存
- 作品データの取得
- 作品の公開設定
- GIF/動画変換
- WebSocket通信
