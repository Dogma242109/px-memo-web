# 要件定義

## 初期スコープ

最初のバージョンでは、バックエンドを使わず、HTML/CSS/JavaScriptのみで動作するWebアプリを目指す。

### 必須機能

- キャンバスにペンで線を描ける
- 消しゴムを使える
- ペン色を変更できる
- ペンサイズを変更できる
- フレームを追加できる
- フレームを削除できる
- フレームを複製できる
- フレームを順番に再生できる
- FPSを変更できる
- 現在のフレーム一覧を確認できる

### できれば入れたい機能

- ひとつ前、二つ前の画像を薄く表示する(設定で変更できるとなおよい)
- カラーは初期は白、黒、赤、緑、青、黄色の5色。
- レイヤーの実装初期3枚
- ブラウザ内に作品を保存できる
- 保存した作品を読み込める
- 作品データをJSONとして書き出せる
- JSONから作品データを読み込める

### 余裕があれば入れたい機能

- PNG連番で書き出せる
- GIFとして書き出せる
- WebMなどの動画形式で書き出せる
- レイヤー機能
- ショートカットキー
- 筆圧風の描画表現

## 将来スコープ

- ユーザー登録
- ログイン
- アカウントごとの作品保存
- クラウド保存
- 複数端末同期
- 作品公開
- 作品共有URL
- コメント
- いいね
- お絵描きチャット
- WebSocketによるリアルタイム描画共有
- サーバー側でのGIF/動画変換
- 設定で変更できることを増やす、レイヤー数、色数など。ただし制限がある中での楽しさもあるのでこれは要検討

## 初期バージョンでは対象外

- 本格的なユーザー認証
- 外部公開用サーバー構築
- スマートフォン完全対応
- 複数人同時編集
- 作品投稿サイト機能
