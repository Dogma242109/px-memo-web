# 要件定義

## 初期スコープ

最初のバージョンでは、バックエンドを使わず、HTML/CSS/JavaScriptのみで動作するWebアプリを目指す。

### 必須機能

- キャンバスにペンで線を描ける
- 消しゴムを使える
- ペン色を変更できる
- ペンサイズを変更できる
- ペン先を変更できる
- パターンブラシを使える
- ペン先形状とパターンを別々に選べる
- 表示倍率を変更できる
- フレームを追加できる
- フレームを削除できる
- フレームを複製できる
- フレームを順番に再生できる
- FPSを変更できる
- Speed 0-10の再生速度プリセットを使える
- 現在のフレーム一覧を確認できる
- 現在のフレームを白紙に戻せる
- Undo/Redoができる
- Ctrl+ZでUndo、Ctrl+Shift+ZでRedoができる

### 初期キャンバス仕様

- 内部描画サイズは310×230pxとする
- 表示倍率は初期値3倍とする
- 表示倍率は2倍、3倍、4倍、5倍から選べる
- 表示時はドットがぼけないようにpixelated表示にする
- 初期ペンはアンチエイリアスのない四角ブラシとする
- 初期段階ではPCブラウザでの利用を優先し、スマートフォン最適化は対象外とする

### できれば入れたい機能

- ひとつ前、二つ前の画像を薄く表示する(設定で変更できるとなおよい)
- カラーは初期は白、黒、赤、青、緑、黄色の6色。

| 色 | 色コード |
| --- | --- |
| 白 | #ffffff |
| 黒 | #000000 |
| 赤 | #ff0000 |
| 青 | #0000ff |
| 緑 | #008a36 |
| 黄 | #fff000 |
- レイヤーの実装初期3枚
- 方眼背景
- ズーム状態で編集できる
- ブラウザ内に作品を保存できる
- 保存した作品を読み込める
- 作品データをJSONとして書き出せる
- JSONから作品データを読み込める

### 余裕があれば入れたい機能

- PNG連番で書き出せる
- GIFとして書き出せる
- WebMなどの動画形式で書き出せる
- レイヤー機能
- ショートカットキー
- 筆圧風の描画表現

## 将来スコープ

- ログイン
- アカウントごとの作品保存
- クラウド保存
- 複数端末同期
- 作品公開
- 作品共有URL
- コメント
- いいね
- お絵描きチャット
- WebSocketによるリアルタイム描画共有
- サーバー側でのGIF/動画変換
- 設定で変更できることを増やす、レイヤー数、色数など。ただし制限がある中での楽しさもあるのでこれは要検討

## 初期バージョンでは対象外

- 本格的なユーザー認証
- 外部公開用サーバー構築
- スマートフォン完全対応
- 複数人同時編集
- 作品投稿サイト機能
